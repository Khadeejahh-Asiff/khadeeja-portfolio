# Maryam Ali - Portfolio Website

Personal portfolio website for Maryam Ali, built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Overview

This portfolio highlights:

- Technical expertise (frontend, backend, architecture, security, performance)
- Featured full-stack and enterprise projects
- Contact channels (email and GitHub)

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Key Sections

- Hero section with updated personal branding
- Expertise section with categorized technical skills
- Projects section with multiple project cards and consistent card sizing
- Contact section (Email + GitHub)

## Featured Projects

- Hiredroid
- ScrumDroid
- HRango
- BankMind.ai
- Salon Reservation System
- 10Marla
- SorrowPlace

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Install and Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
